## Getting Started

You can install this app on your local machine and run it.

### Prerequisites

First install npm.
```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/Cindy1229/COVID-Tracker.git
```
2. Install NPM packages
```sh
npm install
```

3. Start the app
```sh
npm start
```
